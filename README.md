# 🎨 Zamin Graphixs - Graphic Designer Portfolio

A premium, high-performance portfolio website designed and developed for **Zamin Abbas (Zamin Graphixs)**. This project showcases a clean, minimalist aesthetic tailored for a creative professional specializing in high-CTR thumbnail design.

🔗 **Live Demo:** [Zamin Graphixs Portfolio](https://zamingraphixs.netlify.app/)  
🔗 **GitHub Pages:** [AhmedAli-AAC Portfolio Link](https://ahmedali-aac.github.io/Graphic-Designer-portfolio/)

---

## 🚀 About the Project

This portfolio was built to represent **Zamin Abbas**, a Graphic Designer with 3+ years of experience. The goal was to create a visual hook—much like the thumbnails Zamin creates—that connects the designer with potential clients through a seamless user experience.

### Key Features:
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.
- **High-CTR Portfolio Gallery:** A dedicated section showcasing 200+ thumbnails with an interactive layout.
- **Skill Visualization:** Progress bars and tool icons for Adobe Photoshop, Illustrator, Figma, etc.
- **Direct Conversion Path:** Integrated WhatsApp API for instant client-designer collaboration.
- **Modern UI/UX:** Smooth scrolling, clean typography, and a professional color palette.

---

## 🛠️ Tech Stack & Skills Used

As the **Designer and Developer** of this project, I utilized the following technologies:

- **Frontend:** HTML5, CSS3 (Custom Flexbox/Grid layouts), JavaScript.
- **Animations:** CSS Keyframes & AOS (Animate On Scroll) for a dynamic feel.
- **Deployment:** GitHub Pages & Netlify.
- **Performance:** Optimized images and minified assets for lightning-fast loading speeds.
